Global Energy Consumption Analysis (1965–2024)

An interactive Power BI project exploring decades of global energy shifts, from traditional fossil fuels to the rise of renewables.

Technical Implementation
* **Data Modeling:** Integrated global energy datasets (TWh) from 1965 to 2024.
* **DAX Measures:** Calculated renewable energy share %, year-over-year growth, and total consumption metrics.
* **Dynamic Visualization:** * **Area Charts:** Visualized the "Energy Mix" evolution over 50+ years.
  * **Geospatial Analysis:** Implemented both 3D Globe and Azure Map views for regional consumption tracking.
  * **Time Slicers:** Fully animated play-axis for historical trend analysis.

Files in this Repository
* `Global_Energy_3D_Globe`: Full version with 3D modeling.
* `Global_Energy_Web_Accessible`: Web-optimized version.
